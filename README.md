# todo-meteor-react-flow
This is an example of a Todo List with Meteor, React and Flow Router with login,logout and register options

## Documentation
- [Meteor](http://docs.meteor.com/#/basic/)
- [React](https://facebook.github.io/react/docs/getting-started.html)
- [Flow Router](https://kadira.io/academy/meteor-routing-guide/content/rendering-react-components)

## Example
[simple-todos-react](https://www.meteor.com/tutorials/react/creating-an-app)
